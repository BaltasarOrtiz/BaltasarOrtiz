I hold a university-level degree in Programming and I'm an advanced **Computer Science** student. I build modern applications with **Laravel**, **Vue 3**, **Inertia.js**, **Filament**, and **Tailwind CSS**, with a strong interest in scalable architecture and reliable deployments.

- 🔭 **Currently learning:** Advanced Laravel architecture, multi-tenant SaaS patterns, and infrastructure automation.
